# Shuwa Learning Platform

手話学習のための個人用学習プラットフォームです。
動画を登録し、ブックマークやメモを取りながら学習を進められます。

---

## できること

- コース / セクション / レッスンの階層管理
- ローカル動画・YouTube・外部URLを動画ソースとして登録
- A-B リピート（区間繰り返し再生）
- 時刻付きブックマーク・メモ
- 振り返り画面（全レッスンのブックマーク・メモを一覧）
- 学習進捗の記録
- データのJSONエクスポート・インポート（バックアップ）

---

## 推奨ブラウザ

**Chrome または Edge を推奨します。**

- ローカル動画の登録・再生には [File System Access API](https://developer.mozilla.org/docs/Web/API/File_System_Access_API) が必要です（Chrome / Edge のみ対応）
- Safari / Firefox ではローカル動画機能が使えません
- YouTube・外部URL はすべてのモダンブラウザで動作します

---

## 動画ソースの種類と使い分け

| 種類 | 向いている用途 | 制約 |
|------|--------------|------|
| **ローカル** | 手元の動画ファイルを使う | このブラウザ・この端末でのみ再生可 |
| **YouTube** | 複数端末・継続利用・共有 | 限定公開URLも利用可 |
| **外部URL** | 直接再生できるURLがある場合 | URLが有効な間は端末を選ばず再生可 |

### ローカル動画について

- ファイルの関連付けはブラウザの IndexedDB に保存されます
- **別のブラウザ・別の端末では再リンクが必要**です
- バックアップを復元した後もローカル動画は再リンクが必要です
- 複数端末で継続利用したい場合は YouTube 限定公開または外部URLが向いています

> Dropbox / Google Drive の共有ページURLは直接再生できません。
> 直接再生できるダイレクトURLが取得できる場合のみ外部URLとして利用できます。

---

## バックアップについて

学習データ（コース・レッスン・ブックマーク・メモ・進捗）は**このブラウザ内にのみ保存**されます。

- ブラウザのデータ消去・端末変更・OS再インストールでデータが失われます
- **設定 → JSONエクスポート**で定期的にバックアップしてください
- ローカル動画のファイル参照はバックアップに含まれません（YouTube・外部URLは含まれます）

---

## 使い方（最短手順）

1. サイドバーの **「＋」** からコース・セクション・レッスンを作成
2. レッスンを開き **「動画を追加」** から動画を登録
3. 動画を再生しながら、下タブから **ブックマーク・メモ** を記録
4. ナビの **「振り返り」** で記録を復習
5. **設定** から定期的にJSONバックアップ

---

## URLを受け取った方へ

このアプリは**各自のブラウザに学習データが保存される**個人用ツールです。

- アクセスすると最初に使い方ガイドが表示されます
- **Chrome または Edge** でアクセスしてください
- データは共有されません。各自の学習内容は各自のブラウザにのみ保存されます
- ローカル動画を使う場合は、自分の端末でファイルを選択して登録してください

---

## 既知の制約

- ローカル動画はChrome / Edge のみ対応（Safari・Firefoxは非対応）
- データはブラウザ内保存のため、バックアップなしでのブラウザデータ消去はデータ消失につながります
- YouTube埋め込みは利用規約の範囲内でご利用ください
- オフラインでも動作しますが、YouTube・外部URLの再生にはネットワークが必要です

## Vercel へのデプロイ

### 初回

1. このリポジトリを GitHub に push
2. [Vercel](https://vercel.com) でリポジトリをインポート
3. フレームワーク: **Vite** を選択（自動検出されます）
4. ビルドコマンド: `npm run build`
5. 出力ディレクトリ: `dist`
6. **Deploy** を押す

SPAルーティングは `vercel.json` の設定で対応済みです。

### 再デプロイ

`main` ブランチへの push で自動デプロイされます（Vercel のデフォルト設定）。
