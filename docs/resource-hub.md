# Resource Hub — リソースハブ設計

## 概要

学習に使う外部リソース（URL）を一元管理する機能。
「あれどこだっけ？」をなくし、学習に必要なリンクへワンクリックでアクセスできる状態を作る。

## カテゴリ設計

カテゴリはユーザーがUIから自由に作成・編集・削除・並び替えできる。

**推奨初期カテゴリ（ユーザーが最初に作成する）:**

| カテゴリ名 | 用途例 |
|---|---|
| 単語検索 | 手話単語を調べるアプリ・サイト |
| YouTube | 学習に使うチャンネル・プレイリスト |
| 辞書 | 手話辞典・動画辞書 |
| 参考資料 | 技能検定の公式サイト・資料PDF |
| 学習補助 | クイズアプリ・練習ツール |

## 操作フロー

### リソースを追加する
1. リソースハブ画面の「+ リソース」ボタンをクリック
2. ダイアログでタイトル・URL・カテゴリ・説明を入力
3. 「保存」でリソースが一覧に追加される

### リソースを開く
1. ResourceCard の「開く ↗」をクリック
2. 新規タブで URL が開く（`target="_blank" rel="noopener noreferrer"`）
3. `lastAccessedAt` が更新される（将来: 「最近使ったリンク」に反映）

### URL の安全性確認
- `http://` または `https://` 以外のスキーム（`javascript:` 等）を入力した場合、エラーを表示
- 保存時にURLの基本バリデーションを行う

MVP では実装しないが、拡張を見越した設計にしておく。

- お気に入りのみ表示
- 最近使ったリンク順ソート
