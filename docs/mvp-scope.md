# MVP Scope — スコープ定義

## MVP の定義

> アプリを開いたら5秒以内に動画再生が始められ、
> よく使う学習リソースにワンクリックでアクセスできる状態。
> かつ、コース・レッスン・リソースをUIから管理できること。

---

## MVP 機能一覧

### コア: コース・レッスン管理（CRUD）

| 機能 | 詳細 |
|---|---|
| コース作成 | タイトル・説明を入力してコースを作成 |
| コース編集 | タイトル・説明を変更 |
| コース削除 | コース（とその配下のセクション・レッスン）を削除 |
| セクション作成 | コース内にセクションを追加 |
| セクション編集・削除 | セクション名の変更・削除 |
| レッスン作成 | セクション内にレッスンを追加（タイトル・説明） |
| レッスン編集・削除 | タイトル・説明の変更・削除 |
| 並び替え | セクション・レッスンの順序変更（上下ボタン） |

### コア: 動画ソース管理（CRUD）

| 機能 | 詳細 |
|---|---|
| ローカル動画登録 | File System Access API でファイルを選択して登録 |
| リモート動画登録 | 外部URL（mp4等）を入力して登録 |
| YouTube登録 | YouTube URL を入力して登録 |
| 動画ソース変更 | 既存レッスンの動画ソースを差し替え |

### コア: 動画再生

| 機能 | 詳細 |
|---|---|
| ローカル動画再生 | File System Access API 経由でローカルファイルを再生 |
| リモート動画再生 | 外部URL動画を `<video>` で再生 |
| YouTube再生 | YouTube埋め込みで再生（`<iframe>`）|
| 再生・一時停止 | ボタン + スペースキー |
| シークバー | クリック・ドラッグでシーク |
| 音量調整 | スライダー + ミュート |
| 再生速度 | 0.5x / 0.75x / 1x / 1.25x / 1.5x / 2x |
| 5秒スキップ | J/L キー または ボタン |
| 10秒スキップ | Shift+J / Shift+L または ボタン |
| A-Bリピート | A点・B点を設定して区間リピート |
| 再生位置記憶 | レッスン切り替え・再起動後も前回位置から再開 |
| キーボードショートカット | 後述 |

**MVP キーボードショートカット:**

| キー | 操作 |
|---|---|
| `Space` | 再生・一時停止 |
| `←` / `→` | 5秒戻し / 送り |
| `J` / `L` | 5秒戻し / 送り |
| `Shift+J` / `Shift+L` | 10秒戻し / 送り |
| `K` | 一時停止 |
| `A` | A点を設定 |
| `B` | B点を設定 |
| `Esc` | A-Bリピート解除 |
| `[` / `]` | 速度を1段下げ / 上げ |

### コア: リソースハブ（CRUD）

| 機能 | 詳細 |
|---|---|
| カテゴリ作成 | 名前・説明・色でカテゴリを作成 |
| カテゴリ編集・削除 | カテゴリの変更・削除 |
| リソース登録 | タイトル・URL・説明・カテゴリでリソースを登録 |
| リソース編集 | 各フィールドの変更 |
| リソース削除 | リソースを削除 |
| リソースを開く | クリックで新規タブに開く（noopener noreferrer） |
| お気に入り設定 | リソースをお気に入りに設定・解除 |
| カテゴリ別表示 | カテゴリタブで絞り込み表示 |

### コア: データ永続化

- 全データを IndexedDB（Dexie.js）に保存
- アプリ終了・再起動後にデータが保持される
- ローカルファイルハンドルの再取得フロー（Chrome の権限が切れた場合の案内）

## MVP から除外する機能

以下は Phase 4〜9 で対応。MVP では実装しない。

| 機能 | 理由 |
|---|---|
| タイムスタンプ付きメモ | 基本再生が安定してから |
| ブックマーク | 同上 |
| 学習完了・進捗管理 | CRUD と再生の基盤が必要 |
| ホームダッシュボード | UI の全体像が確定してから |
| ダークモード | UIデザイン確定後 |
| レッスン検索 | データが増えてから |
| モバイルレスポンシブ | デスクトップUIを固めてから |
| データエクスポート/インポート | Phase 9 |
| Supabase連携 | Phase 10+ |
| Dropbox等の外部ストレージ | VideoSourceAdapterの枠組みは用意するが実装は後で |

## MVP 完成の受け入れ基準

- [ ] コースを少なくとも1つ作成・表示・削除できる
- [ ] セクションとレッスンを階層的に追加・表示・削除できる
- [ ] ローカル動画ファイルを選択してレッスンに登録し、再生できる
- [ ] 外部URL動画を登録し、再生できる
- [ ] YouTube URLを登録し、埋め込み再生できる
- [ ] 再生位置がレッスン切り替え後も記憶されている
- [ ] A-Bリピートが動作する
- [ ] キーボードショートカットの主要操作が動作する
- [ ] リソースカテゴリを作成・編集・削除できる
- [ ] リソースリンクを追加・編集・削除でき、クリックで新規タブが開く
- [ ] アプリを閉じて再度開いてもデータが保持されている
- [ ] UIからのみ操作でき、JSON編集・フォルダ操作が不要

## 初期データ方針

**サンプルデータは提供しない。**

アプリの初回起動時は空の状態からスタートする。
「データがない状態」に対して分かりやすいエンプティステート（案内メッセージ + 追加ボタン）を表示することで、
UIだけで学習データを育てていける体験を最初から提供する。
