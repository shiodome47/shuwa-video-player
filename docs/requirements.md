# Requirements — Shuwa Learning Platform

## 機能要件

### MVP 必須要件

#### R-01 コース管理
- コースを作成・編集・削除できる（タイトル・説明・カバー画像）
- コース内にセクションを作成・編集・削除・並び替えできる
- セクション内にレッスンを作成・編集・削除・並び替えできる
- すべての操作はアプリ内UIから行える（JSONや外部ツール不要）

#### R-02 動画ソース管理
- レッスンに動画ソースを登録できる
- 対応タイプ:
  - `local`: ローカルファイル（File System Access API経由）
  - `remote`: 外部URL直接参照（mp4等のダイレクトリンク）
  - `youtube`: YouTube URL（埋め込み再生）
- 動画ソースは後から変更・差し替えできる

#### R-03 動画再生
- 対応する全ソースタイプの動画を再生できる
- 再生コントロール: 再生・一時停止・シークバー・音量・再生速度
- 再生速度プリセット: 0.5x / 0.75x / 1x / 1.25x / 1.5x / 2x
- スキップ: 5秒戻し / 5秒送り / 10秒戻し / 10秒送り
- A-Bリピート: 区間を指定して繰り返し再生
- 前回再生位置を記憶し、次回起動時に復元する
- キーボードショートカットで主要操作が可能

#### R-04 リソースハブ
- 学習用外部URLを登録・編集・削除できる
- 各リソースにカテゴリ・タイトル・URL・説明を設定できる
- カテゴリを作成・編集・削除・並び替えできる
- リソースを新規タブで開く
- お気に入り設定ができる
- すべての操作はアプリ内UIから行える

#### R-05 データ永続化
- すべてのデータはブラウザのIndexedDB（Dexie.js）に保存する
- アプリを閉じて再度開いても全データが保持される
- ブラウザのキャッシュクリアでデータが消えないこと（明示的なクリアのみ）

### Phase 4〜6 要件（MVP後）

#### R-06 学習支援
- タイムスタンプ付きメモ（動画の特定時刻にメモを紐づける）
- ブックマーク（動画内の特定時刻にラベル付きブックマーク）
- お気に入り（レッスンをお気に入り登録）
- 学習完了マーク（レッスン単位で完了/未完了を管理）
- 進捗集計（コース単位の完了率）

#### R-07 ホームダッシュボード
- 「続きから学習」（最後に見ていたレッスン）
- 「よく使うリンク」（ピン留めしたリソース）
- 「最近見た動画」
- 「お気に入りレッスン」

#### R-08 モバイル対応
- サイドバーをドロワー化
- タッチ操作でのシーク・コントロール対応
- 動画プレイヤーが縦画面で快適に使える

#### R-09 データ管理
- 全データをJSONにエクスポートできる
- エクスポートしたJSONをインポートできる（バックアップ・復元）

## 非機能要件

### パフォーマンス
- 初回起動（コールドスタート）でレッスン一覧が表示されるまで2秒以内
- レッスン切り替えで動画の再生準備が始まるまで500ms以内
- サイドバーのスクロールが60fps以上

### 信頼性
- データ書き込みはトランザクションで保護し、中断しても不整合が起きない
- ファイルハンドルが失効した場合（File System Access API）、再取得を促すUIを表示する

### 保守性
- データ層（StorageAdapter）と動画参照層（VideoSourceAdapter）はインターフェースで抽象化する
- 将来的に IndexedDB から Supabase に移行する際、ビジネスロジックを変更しない

### アクセシビリティ
- キーボードのみで主要操作が可能
- 動画プレイヤーのコントロールにフォーカスが当たる

### セキュリティ
- 外部URLを開く際は `noopener noreferrer` を付与する
- XSS対策: URLの検証（`javascript:` スキームのブロック）

---

## 制約・前提条件

- 使用ブラウザ: Chrome または Edge（File System Access API必須）
- ローカル動画: ファイルシステム上のmp4/webm/mov等の動画ファイル
- インターネット接続: remote/youtube ソースの再生時のみ必要
- サーバー不要: MVPはピュアブラウザアプリ（バックエンドなし）

---

## スコープ外

- 動画のダウンロード・トランスコード
- 複数ユーザー対応
- 認証・アクセス制御（MVP期間中）
- プッシュ通知・リマインダー
- 手話の認識・採点機能
